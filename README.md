# Inventory Project

This is a full-stack inventory management solution with:

- **Backend**: ASP.NET Core Web API
- **Frontend**: Angular

## Project Structure

inventory-project/
├── inventory-api/ # ASP.NET Core Backend
└── inventory-app/ # Angular Frontend

## How to Run

### Backend
```bash
cd inventory-api
dotnet run
```

### Frontend
```bash
cd inventory-app
npm install
ng serve
```
